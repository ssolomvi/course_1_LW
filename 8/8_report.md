## 8. Распечатка протокола

Программа, вычисляющая сумму двух целых чисел.
```C
#include <stdio.h>
#include <stdlib.h>
#define _CRT_SECURE_NO_WARNINGS

int main(void) {
    int a; int b;
    scanf("%d", &a);
    scanf("%d", &b);
    printf("%d", (a + b));
    printf("\n");
    return 0;
}
```
***
Программа, вычисляющая сумму задаваемого количества целых чисел.
```C
#include <stdio.h>
#include <stdlib.h>
#define _CRT_SECURE_NO_WARNINGS

int main(void) {
    long long a; long long b; long long sum = 0;
    scanf("%lld", &a);
    for (int i = 0; i < a; i++){
        scanf("%lld", &b);
        sum += b;
    }
    printf("%lld", sum);
    printf("\n");
    return 0;
}
```
***
Программа, вычисляющая сумму задаваемого количества целых чисел, больших 10.
```C
#include <stdio.h>
#include <stdlib.h>
#define _CRT_SECURE_NO_WARNINGS

int main(void) {
    long long a; long long b; long long sum = 0;
    scanf("%lld", &a);
    for (int i = 0; i < a; i++){
        scanf("%lld", &b);
        if (b >= 10){
            sum += b;
        }
    }
    printf("%lld", sum);
    printf("\n");
    return 0;
}
```
